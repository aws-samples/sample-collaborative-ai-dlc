# Configuration

!!! abstract "About this section"
    How to configure AIDLC Collaborative for your environment. This section covers environment variables, authentication providers, LLM model settings, and database setup. Most defaults work out of the box for local development, but you will need to adjust these settings when deploying to production or connecting to external services.
